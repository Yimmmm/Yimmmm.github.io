Redis内部实现原理

Posted on 2018-07-30

　　是完全开源免费的，用C语言编写的，遵守BSD协议，
是一个高性能的(key/value)分布式内存数据库，基于内存运行
并支持持久化的NoSQL数据库。

string 字符串

  • string类型是二进制安全的
  • 字符串value最多可以是512M

hash 哈希

zset 有序集合

数据结构

SDS

　　简单动态字符串(Simple Dynamic String） 　　

数据结构

与C字符串相比优势
  • 二进制安全
    • C是通过特殊字符（\0）结束
    • Redis中len属性通过len来
  • 减少内存重分配次数
    • 内存预分配
      • 修改之后的字符串长度<1M,则分配与当前字符串相同的空间
      • 修改之后的字符串长度>1M,则分配空间为1M
    • 惰性释放
      • 当字符串缩短时，空间不会释放，而是标识free
  • 杜绝缓冲区溢出
